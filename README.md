# Diplom_1
Дипломный проект, задание №1: юнит-тесты
Тестируемый сервис - "Stellar Burger"
Ссылка на сервис - https://stellarburgers.nomoreparties.site

В корневой директории проекта содержатся вспомогательные инструменты для тестирования и набор тестов:
1. conftest - файл содержит фикстуры, необходимые для запуска тестов;
2. data_for_test - файл содержит данные для тестов;
3. tests - директория содержит файлы с тестами;
4. praktikum - директория содержит файлы с кодом программы;
5. requirements - в файле перечислены все внешние зависимости тестов;
6. htmlcov - директория содержит файлы с результатами тестового покрытия;
7. README - в файле содержится общая информация по проекту.

В рамках задания созданы юнит-тесты, покрывающие классы `Bun`, `Burger`, `Ingredient`, `Database`
Процент покрытия 100%. Отчет: `htmlcov/index.html`

Для установки зависимостей необходимо в терминал ввести команду: `pip install -r requirements.txt`
Для запуска всех тестов необходимо в терминал ввести команду: `pytest -v`
Для запуска всех тестов и создании HTML-отчета о покрытии необходимо в терминал ввести команду: `pytest --cov=praktikum --cov-report=html`
