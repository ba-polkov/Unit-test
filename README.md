## Дипломный проект. Задание 1: Юнит-тесты

### Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие класс `Burger`

Процент покрытия 100% (отчет: `htmlcov/index.html`)

### Структура проекта

- `practicum` - пакет, содержащий код программы
- `tests` - пакет, содержащий тесты: `test_burger.py`
- `data` - пакет, содержащий тестовые данные
- `conftest` - пакет, содержащий фикстуры

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ pytest --cov=practicum.burger --cov-report=term-html`
