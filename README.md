Тестирование класса Burger
Этот проект содержит тесты для класса Burger из модуля praktikum.
Структура проекта
project/
├── conftest.py
├── test_burger.py
└── README.md

Зависимости
Python 3.x

pytest

unittest.mock

Фикстуры
В conftest.py определены следующие фикстуры:

burger() - создает экземпляр класса Burger

mock_bun() - создает мок-объект булочки

mock_ingredient_filling() - создает мок-объект начинки

mock_ingredient_sauce() - создает мок-объект соуса

multiple_mock_ingredients() - создает несколько мок-объектов ингредиентов

Тесты
В test_burger.py содержатся тесты для следующих методов класса Burger:

Основные методы
set_buns() - установка булочки для бургера

add_ingredient() - добавление ингредиента

remove_ingredient() - удаление ингредиента

move_ingredient() - перемещение ингредиента

get_price() - расчет стоимости бургера

get_receipt() - получение чека

Ключевые тестовые сценарии
Установка булочки - проверка корректного сохранения булочки

Добавление ингредиентов - проверка увеличения количества и порядка хранения

Удаление ингредиентов - проверка уменьшения количества и обработки ошибок индекса

Перемещение ингредиентов - проверка изменения порядка

Расчет стоимости - тестирование различных комбинаций булочек и ингредиентов

Формирование чека - проверка включения всех необходимых данных в чек

Обработка ошибок - тестирование исключительных ситуаций

Для запуска тестов выполните команду:
pytest test_burger.py -v

Особенности тестирования:
Используются мок-объекты для изоляции тестов

Параметризованные тесты для проверки различных ценовых сценариев

Проверка вызовов методов мок-объектов

Тестирование граничных случаев и обработки ошибок
