Процент покрытия 100% (отчет: `htmlcov/index.html`)

### Структура проекта

- `Diplom_1` - корневая папка с файлами по выполнению задания первой части автоматизации
- `tests` - пакет, содержащий тесты класса `Burger` и файл с фикстурами
- `.gitignore` - файл для игнорирования
- `bun.py` - исходный код класса Bun
- `burger.py` - исходный код класса Burger
- `data.py` - файл со вспомогательными данными
- `database.py` - исходный код класса Database
- `ingredient.py` - исходный код класса Ingredient
- `ingredient_types.py` - перечисление типов ингредиентов
- `praktikum.py` - исходный код задания
- `README.md` - файл с описанием проекта
- `requirements.txt` - файл c указанием внешних зависимостей

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ pytest --cov=tests --cov-report=html`
