## Задание 1: Юнит-тесты

### Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

- Созданы юнит-тесты, покрывающие класс `Burger` с 100% покрытием
- Использованы моки для классов `Bun` и `Ingredient`
- Применена параметризация для тестов цены и генерации чека
- Тестовые данные вынесены в отдельный модуль `data.py`
- Фикстуры вынесены в `conftest.py` для переиспользования
- Все 12 тестов проходят успешно

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ pytest --cov=praktikum --cov-report=html`
