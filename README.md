## Дипломный проект. Задание 1: Юнит-тесты

### Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие классы `Bun`, `Burger`, `Database`, `Ingredient`

Применены техники параметризации и мокирования

Процент покрытия - 78% (отчет: htmlcov/index.html)

### Структура проекта

Проект выполнен с использованием фреймворка pytest

В директории praktikum нахдится пакет, содержащий исходный код программы

В директории tests находится пакет, содержащий юнит-тесты, разделенные по классам:
- файл test_bun.py содержит класс `TestBun` с тестами для покрытия класса `Bun`
- файл test_burger.py содержит класс `TestBurger` с тестами для покрытия класса `Burger`
- файл test_database.py содержит класс `TestDatabase` с тестами для покрытия класса `Database`
- файл test_ingredient.py содержит класс `TestIngredient` с тестами для покрытия класса `Ingredient`

Файл data.py содержит класс `Data` со статическими данными для юнит-тестов

Файл conftest.py содержит pytest-фикстуры для инициализации моков

Файл requirements.txt содержит версии пакетов, необходимых для работы проекта

### Запуск автотестов

**Установка зависимостей**

> pip install -r requirements.txt

**Запуск автотестов и создание HTML-отчета о покрытии**

> pytest --cov=praktikum --cov-report=html
